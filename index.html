AndroPred

AndroPred is a Deep Learning Based System for Android Malware Detection.

Android is the most used mobile operating system in the world. Due to its popularity, malware is steadily increasing every year, posing many problems for users. As malware has increased, anti-malware solutions have as well.

Nowadays, a large number of researchers and companies began research on android malware detection.

Research studies in the Android malware detection field work in three approaches static, dynamic or hybrid. In static analysis, malware is disassembled into a source code from where specific features are extracted. In dynamic analysis, malware is monitored at runtime in a virtual environment. In both approaches, machine learning algorithms have been used to build classification models by training classifiers with dataset of malware features that collected from static or dynamic analysis. The learning classification models are then used to detect malicious apps and classify them into their families.

Deep learning, as a new area of machine learning research, has gained increasing attentions in artificial intelligence (AI) and motivated a great number of successful applications in speech and image recognition.

In this work in order to establish an extensive research about how machine learning and deep learning are used to detect malware in Android platform, we discuss relevant background related to the Android OS, machine learning and Deep learning. Then we discuss he different approaches to detect Android malware based ML and DL.

In order to protect the security of android system, we proposed an intelligent malware detection method based on deep learning.

Based on solution proposed by this articles [Karbab et al 2017], [Liang et al 2017] and [Ding et al 2020] which they used bytecode images techniques and text categorization for manifest. We proposed our solution which is a CNN model to classify android application to malware or benign using AndroidManifest.xml bytecode images dataset as input to the model.

Network Security Automation

With the evolution of telecommunications services and multimedia data traffic, Cyber-attacks are becoming day-to-day more stronger ,lethal and frequent, due to the fact that most companies have poor cybersecurity practises, and the exis- tence of ready hacking tools that don’t require knowledge or skills to use them in order to achieve their different objectives like accessing, modifying, or destroying sensitive information, stealing money from users or disturbing and interrupting business functions.

The network security administrators need to update the routers, firewalls and other devices configurations to maintain the cadence and safety of the enterprise activities, and despite the fact that these updates can be critical for the network functioning and security, the risk of their implementation is increased due to the human mistake factor and how the process can be daunting.

One of the rising technologies in network automation is Software Defined Network (SDN) that decoupled the control plan from the data plan which resulted in providing a flexible and powerful network architecture for network application such as network management, QoS, load balancing and content delivery system. Regardless of the recent numerous uses of SDN in networking, the security community is relatively slow in adapting it.

In our work we have established an extensive research about how SDN is used to enhance the network security by analyzing the different approaches proposed by the researches in this field, thereafter we will exploit their results to develop a new framework that automate the processes of the network security assessment, design, deployment of the corrective actions.
